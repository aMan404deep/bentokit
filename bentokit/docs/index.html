🍱 BentoKit

A lightweight, flexible Bento-style UI library for HTML, CSS & React.

🔧 Installation

React:

npm install bentokit

HTML/CSS:

<link rel="stylesheet" href="bentokit.css" />

📦 Components

  • BentoCard
  • BentoGrid
  • BentoButton
  • BentoImage

🎨 Theming

Override CSS variables to apply your own theme:

:root { --bento-primary: #f0a; }
